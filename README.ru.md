# 🎮 DeepRenPyTrans

**Автоматический универсальный переводчик визуальных новелл на движке Ren'Py с помощью ИИ.**

🌐 [English](README.md) | [Русский](README.ru.md) | [Español](README.es.md) | [Português](README.pt.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [简体中文](README.zh.md)

---

Переводите любые игры на движке Ren'Py на любой язык, используя DeepSeek (включая модели `deepseek-v4-flash` / `deepseek-v4-pro`), OpenAI или локальные LLM — без изменения исходного кода игры.

---

## ✨ Возможности

- **🔍 Умное извлечение** — автоматически находит все переводимые строки в файлах `.rpy`, отфильтровывая системный код, идентификаторы и отладочные сообщения.
- **🤖 Перевод с помощью ИИ** — пакетный перевод через DeepSeek, OpenAI или локальную Ollama с учетом контекста сцены/главы.
- **🔌 Инъекция во время выполнения** — система хуков с нулевым изменением исходных файлов, использующая встроенную функцию Ren'Py `config.replace_text`.
- **📊 Аудит качества** — поиск непереведенных строк, устаревших переводов и мусорных записей.
- **🧹 Очистка словаря** — автоматическое удаление отладочных строк, строк документации и программных артефактов.
- **⚡ Инкрементальность** — возможность возобновить прерванный перевод, обрабатывая только новые или отсутствующие строки.
- **📱 Кроссплатформенность** — работает со сборками для ПК, Android (инъекция в APK) и iOS.

## 📖 Команды

### `extract` — поиск строк для перевода

```bash
python -m deeprenpytrans extract --game ./MyGame/game --output strings.json
```

Обходит все файлы `.rpy`, находит строки в кавычках и применяет умные фильтры для исключения:
- Внутренних идентификаторов (`ITM_Sword`, `LOC_Bridge`, `ACT_NPC01`)
- Кода Python и проверок условий
- Путей к файлам и HEX-кодов цветов
- Отладочных и лог-сообщений
- Уже переведенного текста (детектор кириллицы/символов)

Параметры:
| Флаг | Описание |
|------|-------------|
| `--game PATH` | Путь к папке `game/` |
| `--output FILE` | Выходной JSON-файл (по умолчанию: `strings_by_file.json`) |
| `--include-log PATH` | Слияние со строками из `untranslated.log` |

### `translate` — перевод через ИИ

```bash
python -m deeprenpytrans translate --strings strings.json --dict dictionary.json
```

Отправляет строки ИИ умными пакетами (группируя по исходным файлам для сохранения контекста).
Поддерживает инкрементальный перевод — уже переведенные строки пропускаются автоматически.

### `audit` — проверка качества перевода

```bash
python -m deeprenpytrans audit --dict dictionary.json --strings strings.json
```

Создает отчет, показывающий:
- ❌ Непереведенные строки (есть в источнике, но нет в словаре)
- 👻 Устаревшие переводы (есть в словаре, но больше нет в игре)
- 🔁 Одинаковые пары ключ=значение (возможно, забыли перевести)
- 📭 Пустые переводы
- 🗑️ Мусорные записи (отладочные сообщения, куски кода)

### `clean` — очистка словаря от мусора

```bash
python -m deeprenpytrans clean --dict dictionary.json --dry-run
python -m deeprenpytrans clean --dict dictionary.json --remove-orphaned
```

Параметры:
| Флаг | Описание |
|------|-------------|
| `--dry-run` | Предварительный просмотр того, что будет удалено |
| `--keep-junk` | Не удалять отладочные/технические строки |
| `--remove-orphaned` | Также удалить строки, не найденные в игре |

### `inject` — генерация hooks.rpy

```bash
python -m deeprenpytrans inject --game ./MyGame/game --lang russian
```

Создает файл `hooks.rpy`, который:
- Загружает `dictionary.json` при запуске игры
- Перехватывает весь текст через встроенный `config.replace_text`
- Записывает непереведенные строки в `untranslated.log` прямо во время игры
- Добавляет горячую клавишу для включения/выключения перевода "на лету"
- Заменяет шрифты игры на поддерживающие ваш язык

## ⚙️ Конфигурация

### `config.yaml`

```yaml
game_dir: "./MyGame/game"
target_language: "Russian"
translation_dir: "russian"

api:
  provider: "deepseek"    # или "openai", "openrouter", "groq", "nebius", "deepinfra", "gemini", "dashscope", "ollama"
  model: "deepseek-chat"  # поддерживает новые deepseek-v4-flash / deepseek-v4-pro
  temperature: 0.2
  batch_size: 40

fonts:
  default: "DejaVuSans.ttf"
  replacements:
    "OriginalFont.ttf": "DejaVuSans.ttf"

extraction:
  skip_prefixes: ["ITM", "ACT", "LOC", "QST"]
  force_include: ["Q.Save", "Q.Load"]
```

### `.env`

```bash
DEEPSEEK_API_KEY=sk-your-key-here
# или
OPENAI_API_KEY=sk-your-openai-key
```

> [!NOTE]
> Поддержка локальных моделей (Ollama) и других API-провайдеров (OpenAI, OpenRouter, Groq, Nebius, DeepInfra, Gemini, DashScope), кроме DeepSeek, является номинальной. Проект разрабатывался под DeepSeek, а остальные интеграции сделаны по стандартам OpenAI API без глубокого практического тестирования. В будущем, по мере возможности, список протестированных ключей будет расширяться.

## 📱 Мобильное развертывание

### Android (Инъекция в APK)
1. Используйте наш автоматический скрипт `build_apk.bat`, который берет на себя всю работу:
   - Распаковка старого APK.
   - Восстановление уже сжатых ассетов из оригинального APK для экономии до 60% размера (в среднем экономится ~400-500 МБ).
   - Конвертация звуков wav-в-ogg.
   - Оптимизация новых картинок под мобильные с пропуском уже сжатых.
   - Сборка APK с ультра-сжатием и его автоматическое подписание.
2. **Настройка параметров**: Вы можете изменить процесс сборки, отредактировав флаги в верхней части файла `build_apk.bat`:
   - `RESTORE_OLD_ASSETS` (1/0): включить/выключить восстановление сжатых ассетов из старого APK.
   - `COMPRESS_AUDIO` (1/0): включить/выключить конвертацию wav-в-ogg и автозамену путей в скриптах.
   - `COMPRESS_IMAGES` (1/0): включить/выключить оптимизацию новых картинок.
   - `INJECT_TRANSLATION` (1/0): включить/выключить инъекцию перевода. Установите `0`, чтобы получить чистую оригинальную (английскую) версию игры на Android.
   - `LANG_FOLDER`: имя папки перевода внутри `game/tl/` (например, `russian`).
   - `COMPRESSION_LEVEL` (0-9): уровень сжатия 7-Zip (9 — максимальное сжатие, 0 — без сжатия).
3. При ручной сборке: распакуйте APK с помощью 7zip, замените файлы внутри `assets/x-game/game/`, удалите подписи из `META-INF/` и пересоберите/подпишите APK.

### iOS
1. Сгенерируйте проект Xcode через Ren'Py Launcher.
2. Добавьте файлы перевода `tl/russian/` в проект.
3. Соберите и установите на устройство.
## ⚠️ Важное примечание и ограничения

- **Статус тестирования**: На данный момент инструмент протестирован и успешно проверен только на одной игре, где все тексты были корректно переведены и собраны в рабочий APK.
- **Специфика кода игр**: Хотя целью было создание идеального универсального переводчика, другие новеллы Ren'Py могут (и, скорее всего, будут) требовать индивидуальной настройки под структуру их кода, кастомные префиксы или особенности скриптов.
- **Не умеете кодить?**: Если вы не умеете программировать или вам просто лень копаться в коде, настоятельно рекомендуем использовать ИИ-ассистентов разработчика вроде **Antigravity**, **Cursor** и других подобных инструментов — они помогут быстро адаптировать экстрактор и фильтры под конкретную игру.
